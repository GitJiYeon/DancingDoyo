# 🐦 Dancing Doyo

> 멧도요새의 특징에서 영감을 받아 제작한 Python 리듬 게임
> 음악의 박자에 맞춰 노트를 입력하고, 새들과 배틀하며 최고의 댄서가 되는 컨셉의 게임입니다.

## 📌 프로젝트 소개

**Dancing Doyo**는 SNS에서 춤추는 멧도요 영상을 보고,
멧도요새 특유의 몸짓과 박자감에서 영감을 받아 제작한 리듬 게임입니다.

플레이어는 음악의 BPM에 맞춰 등장하는 화살표 노트를 입력하며,
다른 새들과 배틀하여 최고의 댄서가 되는 것을 목표로 합니다.

단순히 노트를 입력하는 기능뿐만 아니라,
캐릭터 애니메이션, 판정 이미지, 배경음악, 효과음, 카메라 줌 연출을 활용하여
시각적·청각적 재미를 함께 느낄 수 있도록 제작했습니다.

---

## 🎯 기획 의도

SNS에서 춤추는 멧도요 영상을 접한 뒤,
멧도요새의 독특한 움직임과 리듬감이 리듬 게임과 잘 어울린다고 생각했습니다.

그래서 멧도요가 다른 새들과 배틀하며 최고의 댄서가 된다는 컨셉으로
**Dancing Doyo**를 기획하게 되었습니다.

리듬 게임의 핵심인 타이밍 입력과 판정 시스템을 구현하는 것뿐만 아니라,
게임의 분위기와 연출을 통해 사용자가 더 몰입할 수 있는 경험을 만드는 것을 목표로 했습니다.

---

## 🎮 주요 기능

### 화살표 노트 싱크

* 음악의 BPM에 맞춰 화살표 노트가 등장
* 방향키 입력을 통해 노트 판정
* 노트 타이밍 데이터를 `txt` 파일로 관리
* 노트 차트를 코드와 분리하여 유지보수가 쉽도록 구현

### 판정 시스템

입력 타이밍에 따라 서로 다른 판정이 표시됩니다.

* PERFECT
* GREAT
* GOOD
* NORMAL

정확한 타이밍에 입력할수록 더 높은 점수를 획득할 수 있습니다.

### 콤보 및 점수 시스템

* 연속 입력 성공 시 콤보 증가
* 콤보에 따른 점수 보너스 적용
* 최종 점수, 성공 횟수, 최대 콤보, 미스 횟수 표시
* 미스 횟수에 따른 게임 오버 처리

### 역동적인 카메라 무빙

* 상대의 노트가 올라올 때는 상대를 중심으로 화면 포커스 이동
* 플레이어의 노트가 올라올 때는 플레이어 쪽으로 화면 줌인
* 리듬 배틀 느낌을 살리기 위한 시점 전환 연출 구현

### 캐릭터 애니메이션

* 멧도요와 다른 새 캐릭터를 활용한 게임 화면 구성
* GIF 이미지를 사용한 캐릭터 움직임 표현
* 입력 방향과 게임 상황에 따라 캐릭터 동작 변화

### 로그인 및 랭킹

* Guest 플레이 지원
* 회원가입 및 로그인 기능
* 로그인한 사용자의 최고 점수 저장
* 랭킹 화면에서 상위 점수 확인 가능

## 🎲 게임 진행 방식

1. 메인 화면에서 게임을 시작합니다.
2. Guest 또는 Login 방식을 선택합니다.
3. 음악에 맞춰 등장하는 방향키 노트를 입력합니다.
4. 입력 타이밍에 따라 판정과 점수가 부여됩니다.
5. 콤보를 유지하면 더 높은 점수를 얻을 수 있습니다.
6. 곡이 끝나면 최종 결과 화면이 표시됩니다.
7. 로그인한 사용자는 점수가 저장되며 랭킹에서 확인할 수 있습니다.
